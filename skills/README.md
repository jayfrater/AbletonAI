# Agent Skills Directory

This directory contains standalone skill definitions for the agent, organized by domain. These definitions are designed to be portable and provide a clear specification of the agent's capabilities in specific areas.

## Directory Structure

- **`computer_use/`**: Foundational skills for interacting with an operating system, managing files, and navigating the web.
    - `shell_power_user.md`: Advanced terminal and scripting.
    - `file_system_architect.md`: File and data organization.
    - `web_navigation_expert.md`: Research and browsing proficiency.
- **`ableton/`**: Specialized skills for music production and control of Ableton Live 12.
    - `ableton_maestro.md`: General control and composition.
    - `ableton_assistant.md`: Troubleshooting and help.
    - `ableton_knowledge_base.md`: Manual querying and documentation.
    - ... and other specialized domain skills.

## How to Use These Skills

These Markdown files serve as "System Instructions" or "Skill Profiles" for the agent. They can be:
1.  **Read by the Agent**: To understand its own constraints and specialized knowledge in a given domain.
2.  **Used for RAG**: As source documents for a Retrieval-Augmented Generation system.
3.  **Standalone Documentation**: To show users what the agent is capable of.
