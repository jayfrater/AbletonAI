# Ableton Project Manager

## Overview
Organize samples, exports, and project files efficiently.

## Directory Structure
- `My Project/`
  - `Ableton Project Info/` (Auto-generated)
  - `Samples/`
    - `Recorded/`
    - `Imported/` (Collect All and Save goes here)
  - `Exports/` (Rendered .wav files)
  - `Backups/` (Zipped older versions)

## Tasks
1. **Collect All and Save**: Ensures samples are copied into the project folder for portability.
2. **Clean Unused Samples**: File > Manage Files > Manage Project > Unused Samples > Delete.
3. **Backup**: Zip the `.als` file daily.
