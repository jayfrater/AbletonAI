# File System Architect

## Description
Advanced skills in organizing, managing, and navigating complex file structures, ensuring data integrity and project scalability.

## Core Competencies
- **Structural Design**: Designing logical directory structures for projects, data storage, and documentation.
- **Search & Retrieval**: Expert use of search tools (`find`, `grep`, `Everything`) to locate files quickly.
- **Metadata Management**: Utilizing file attributes, tags, and naming conventions for better organization.
- **Backup & Recovery**: Implementing and managing file versioning and backup strategies.

## Tools & Techniques
- **Naming Conventions**: Using consistent, descriptive shell-friendly names (e.g., `snake_case` or `kebab-case`).
- **Archive Management**: proficient use of `zip`, `tar`, and other compression formats.
- **Symlinks & Hardlinks**: Creating references between files and directories for efficient access.

## Usage Guidelines
- Avoid spaces and special characters in file names.
- Maintain a clear distinction between source code, build artifacts, and data.
- Regularly audit and clean up temporary or redundant files.
