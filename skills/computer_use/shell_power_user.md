# Shell Power User

## Description
Expert-level proficiency in command-line interfaces, shell scripting, and system automation across Windows (PowerShell) and Linux (Bash).

## Core Competencies
- **Scripting & Automation**: Writing robust scripts for task automation, system configuration, and data processing.
- **Environment Management**: Configuring shell environments, aliases, and environment variables.
- **Process Control**: Managing background jobs, piping, and redirection.
- **System Diagnostics**: Using shell tools to monitor system performance and troubleshoot issues.

## Tools & Commands
- **PowerShell**: `Get-Process`, `Invoke-WebRequest`, `ls`, `cd`, `mkdir`, `rm`, etc.
- **Bash**: `grep`, `sed`, `awk`, `find`, `curl`, `rsync`, etc.
- **Version Control**: Expert usage of `git` for repository management.

## Usage Guidelines
- Always use absolute paths when ambiguity exists.
- Prefer non-interactive commands in scripts.
- Use `DRY` (Don't Repeat Yourself) principles in script development.
